# 提权兼容契约

版本: v0.5.0  
日期: 2026-03-28

## 目标

保持当前“必要时可通过 UAC 获得与游戏同层级显示能力”的用户预期，同时避免把所有新逻辑都绑定成永久管理员进程。

## 契约

- `AspenBurner.exe` manifest 默认 `asInvoker`
- 旧 `Start/Configure/Stop` 入口仍可触发提权链
- 若主实例已提升，兼容入口也应以可通信的权限级别发命令
- `Stop` 不能再通过扫描并强杀所有匹配 `powershell.exe` 来完成

## 不再允许的行为

- 用“外部强杀 PowerShell 进程”代替正式实例管理
- 低权限实例对高权限主实例静默失败
