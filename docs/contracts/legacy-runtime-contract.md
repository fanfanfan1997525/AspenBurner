# 旧运行时兼容契约

版本: v0.5.0  
日期: 2026-03-28

## 配置路径

- 主配置文件继续使用 `config/crosshair.json`

## 默认行为

- 默认准心颜色：`Green`
- 默认长度：`6`
- 默认间距：`4`
- 默认粗细：`2`
- 默认描边：`1`
- 默认透明度：`255`
- 默认状态角标：关闭

## 准心显示规则

- 仅当目标进程位于前台时显示
- 当前默认目标进程：
  - `DeltaForceClient-Win64-Shipping`
  - `delta_force_launcher`

## 兼容边界

- 旧配置中的合法 `OffsetX/OffsetY = -4000..4000` 必须继续支持
- 不允许设置器静默夹断合法旧值
