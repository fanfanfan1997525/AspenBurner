# AspenBurner v0.6.0

发布日期: 2026-03-29

## 亮点

- 新增 Windows 安装器与 portable 发布链。
- 兼容入口现在支持安装目录直接运行，不再依赖仓库结构。
- 固化版本元数据，可直接打 tag 与生成 Release 资产。
- 修复颜色、透明度等样式参数修改后真实 overlay 不立即刷新的回归。

## 资产

- `AspenBurner-v0.6.0-win-x64-setup.exe`
- `AspenBurner-v0.6.0-win-x64-portable.zip`

## 验证

- `dotnet test AspenBurner.sln`
- `Invoke-Pester tests/pester/*.ps1`
- `dotnet build src/AspenBurner.App/AspenBurner.App.csproj -c Release`
- 安装器静默安装/启动/停止/卸载 smoke
