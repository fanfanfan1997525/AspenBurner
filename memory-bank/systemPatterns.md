# 系统模式

版本: v0.1.0

- 入口脚本 `CrosshairOverlay.ps1`
  负责加载配置、初始化 WinForms、创建透明点击穿透的小尺寸准心窗口，并按前台窗口状态决定显示/隐藏。
- 核心模块 `src/CrosshairOverlay.Core.psm1`
  负责默认配置、配置合并、参数校验、颜色解析、十字几何坐标计算，以及小尺寸覆盖层边界计算。
- 测试目录 `tests/tdd/crosshair_overlay`
  负责对核心配置、几何逻辑、覆盖层窗口尺寸与前台进程匹配逻辑做 Pester 回归验证，避免后续改动破坏“小十字、无中心点、非桌面常驻”的基本行为。
