# crosshair_overlay

版本: v0.1.0

## 概述

验证中心十字准心的默认配置、参数校验和几何坐标是否稳定，重点保证“小十字、无中心点、绿色/黄色可切换”三件事不会回归。

## 背景与动机

- 问题描述: 需要一个独立的桌面准心覆盖层，而不是依赖游戏内准心开关。
- 创建原因: 防止后续改动把中心空隙、颜色限制或尺寸约束写坏。
- 关联 Issue/PR: 无

## 测试目标

验证配置默认值、配置合并、非法配置拦截、颜色解析，以及四段式十字的几何对称性和中心留空行为。

## Determinism & Tolerance

- 随机性: 无
- 排序: 线段顺序固定为 `Left, Right, Top, Bottom`
- 浮点: 无，全部使用整数坐标
- 失败信号: PowerShell 异常
